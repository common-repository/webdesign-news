== Description ==

Adds a customizeable widget which displays the latest news by [http://www.wdee.de/](http://www.wdee.de/)

*Feature List*

* customizable widget
* displays a user-defined number of links
* title shortening

== Installation ==

1. Extract `webdesign-news.zip`.
2. Upload the `webdesign-news`-folder to your `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Add the widget to your sidebar from Appearance->Widgets and configure the widget options.

*German*
a. Entpacke `webdesign-news.zip`.
b. Lade den `webdesign-news`-Ordner in dein Verzeichnis `/wp-content/plugins` hoch.
c. Aktiviere das Plugin im 'Plugins'-Men&uuml; der Wordpress-Administration.
d. F&uuml;ge das Widget unter Design->Widgets deiner Sidebar hinzu und bearbeite die Widget-Optionen.
